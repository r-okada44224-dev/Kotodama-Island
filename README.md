# 🏝️ ことだまの島 ～まなびの地図～

プログラミングを「ことだま」として体験する**学習RPGゲーム**です。  
HTML → CSS → JavaScript を、知識ゼロから「体で」理解できます。

> 親プロジェクト「（仮称）暗号史RPG ／ Code of History」の**プロローグ学習シリーズ**です。

## 🎮 遊び方

1. `index.html`（まなびの地図）を開いて章を選ぶ
2. **精霊のセリフ**をメッセージウィンドウで読む（クリックで進む）
3. **左のエディタ**にコードを書く
4. **「✨ となえる」**ボタンを押すと実行・判定
5. 正解すると「実はこれは〇〇でした！」と種明かし → 次のクエストへ

### ⌨️ キーボードショートカット
| キー | 動作 |
|------|------|
| `Ctrl + Enter`（Mac: `Cmd + Enter`） | となえる（実行） |
| `Tab`（JS編のみ） | インデント（スペース2つ）挿入 |
| メッセージウィンドウをクリック | セリフを進める |

### 🆘 困ったときは
- **不正解のとき** → 精霊が「どこが違うか」を具体的に教えてくれる
  （例：「おわりタグ `</html>` が みつからないぞ」※CSS/JS編の本編は一部一般メッセージ）
- **💡 ヒント** → 正しいコードのお手本が見られる
- **📖 ずかん**（全章） → クリア済みクエストの「ことだま」の解説をいつでも見返せる。
  下部に「MDNでの調べ方」の案内つき（大全にも「しらべかたの こころえ」の節あり）
- **↺ もどす** → エディタを最初の状態に戻す
- **エラーメッセージ**（JS編・DOM編）→ 日本語でやさしく表示

### 🔥 とっくんモード（全章）
各章の本編をクリアすると解放される復習用の総合問題（各3問）。
「📖 ずかん」で学んだことを確認しながら、白紙から書けるかを試す。
進み具合は章ごとに自動セーブされる。

### ⚙️ せってい・書きかけ保存・にがてメモ
- **🔊 おと / 💬 はやさ** — ゲーム中のステータスバーで効果音ON/OFFとセリフ速度
  （ふつう／はやい／いっき）を切り替えられる。設定は全章共通で保存
- **書きかけの自動保存** — エディタの内容はクエストごとに自動保存。
  途中でページを閉じても、もどってくると書きかけから再開できる（クリアで消える）
- **にがてメモと励まし** — つまずいた回数を記録。同じ問題で3回続けてまちがえると
  精霊がやさしく励まし、「ことだま大全」によくつまずいた問題の一覧が出る

### 📖 ことだま大全（`taizen.html`）
全4章の図鑑を1ページに集めたチートシート。クリアした分だけ解放され、
「🖨 いんさつする」で紙のチートシートにもできる（未解放と にがてメモは印刷されない）。

### ✏️ じゆうちょう（`jiyucho.html`）
学んだ HTML・CSS・JS を**なんでも自由に書ける**制作スペース。
- 書くとすぐ見た目が変わるライブプレビュー（書きかけの `<script>` は暴発しないよう「✨ うごかす」で実行）
- テンプレート（白紙／自己紹介／動くボタン）から始められる
- 自動保存＆「💾 もちかえる」で .html ファイルとしてダウンロード

### 💾 セーブと持ち帰り
- 進み具合は **自動セーブ**（localStorage）。タイトル画面に「▶ つづきから」が出る
- ホームの「まなびの地図」に各章のクリア状況バッジが表示される
- HTML編クリア後、**「💾 まきものを もちかえる」** で自分の書いた自己紹介ページを
  本物の `.html` ファイルとしてダウンロードできる（ダブルクリックでブラウザで開ける！）

## 🛠️ 技術的な特徴

| 機能 | 実装方法 |
|------|---------|
| 純粋JS・フレームワーク不使用 | HTML/CSS/JS のみ |
| **具体的な間違い指摘（HTML編）** | `diagnose` 関数が「どこが違うか」を日本語で返す |
| **ことだま図鑑（HTML編）** | クリア済みクエストの解説をモーダルで一覧表示 |
| **とっくんモード（HTML編）** | 本編クリア後に解放される復習用総合問題 |
| **進捗の自動セーブ** | localStorage（章ごとにキーを分離） |
| **作品の持ち帰り** | Blob + `a[download]` で完成ページを .html 保存 |
| HTML編のリアルタイムプレビュー | `<iframe>` に直接書き込み |
| CSS編のリアルタイムプレビュー | `<iframe>` テンプレートにCSSを注入 |
| **JS編の安全な実行** | `new Function()` + `"use strict"` サンドボックス |
| **DOM編のライブ実行** | プレイヤーのJSを iframe 内で実行、クリック再現で動作判定 |
| **シンタックスハイライト** | 透明textarea＋色付きオーバーレイの自作実装（外部ライブラリなし・HTML/CSS/JS/混在対応） |
| **クリア時の紙吹雪** | モーダル出現を MutationObserver で検知して自動発射（reduced-motion 対応） |
| **記号クイックバー** | タッチ端末・狭い画面で `< > { } \`` などをワンタップ入力 |
| **console.log の横取り** | fakeConsole で出力を配列に収集 |
| **エラーの日本語変換** | エラーメッセージを日本語に翻訳して表示 |
| タイプライター演出 | `setInterval` で1文字ずつ表示 |
| 効果音 | Web Audio API でその場生成 |
| 行番号表示（JS編） | textarea と同期したdiv |
| Tabキーでインデント | `keydown`イベントでスペース2つ挿入 |
| モバイル対応 | 900px以下で縦並びレイアウト |

## 💾 データ管理

サーバー側のデータ保存は一切行いません。  
進行状態はブラウザの `localStorage` に自動保存されます。

| キー | 内容 |
|------|------|
| `kotodama_html_v1` | HTML編のクリア数・特訓クリア数・完成した自己紹介ページのコード |
| `kotodama_css_v1` | CSS編のクリア数・特訓クリア数 |
| `kotodama_js_v1` | JS編のクリア数・特訓クリア数 |
| `kotodama_dom_v1` | DOM編のクリア数・特訓クリア数・完成したメモ帳アプリのコード |
| `kotodama_layout_v1` | 第4章のクリア数・特訓クリア数 |
| `kotodama_settings_v1` | 効果音ON/OFF・セリフ速度（全章共通） |
| `kotodama_draft_<章>` | 書きかけコードの下書き（クエストごと・クリアで削除） |
| `kotodama_mistakes_v1` | つまずき回数の記録（大全の「にがてメモ」用） |
| `kotodama_jiyucho_v1` | じゆうちょうの作品（自動保存） |
